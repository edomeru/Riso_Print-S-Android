■Préface

RISO PRINT-S est une application qui permet d'imprimer le répertoire de documents PDF à partir des tablettes et smartphones.

Vous pouvez, bien entendu, imprimer des documents PDF enregistrés sur vos appareils et configurer plusieurs paramètres d'impression tels que l'impression recto/verso, l'imposition, la perforation et l'agrafage à l'aide de RISO PRINT-S.

Ce guide décrit comment installer, utiliser et configurer cette application, ainsi que certaines restrictions. Lisez-le avant/pendant l'utilisation de cette application ou lorsque vous avez besoin d'informations. Il vous aidera à profiter longtemps de votre application RISO PRINT-S.

■Installation de RISO PRINT-S

Installez RISO PRINT-S sur votre tablette ou smartphone.

La méthode d’installation varie en fonction du système d'exploitation de votre périphérique.

Pour les utilisateurs iPhone ou iPad :

  1. 1.Ouvrez [App Store] sur votre iPhone ou iPad, puis recherchez [RISO PRINT-S].

  2. 2.Appuyez sur [RISO PRINT-S] dans les résultats de recherche, puis appuyez sur le bouton [Gratuit].

    Le bouton [Gratuit] est remplacé par le bouton [Installer].

  3. 3.Appuyez sur le bouton [Installer].

    L'écran [Connexion Apple ID] apparaît.

  4. 4.Entrez votre identifiant et votre mot de passe Apple.

    L’installation commence. Lorsqu’elle est terminée, le bouton [Installer] est remplacé par le bouton [Ouvrir], puis l’icône [] (RISO PRINT-S) est ajoutée dans l'écran d'accueil.

    Lorsque vous installez RISO PRINT-S sur plusieurs périphériques avec le même identifiant Apple, le bouton [] s'affiche en lieu et place du bouton [Gratuit] et l'installation commence automatiquement.

Pour les utilisateurs de périphérique Android :

  1. 1.Ouvrez [Google PlayTM] sur votre périphérique, puis recherchez [RISO PRINT-S].

  2. 2.Appuyez sur [RISO PRINT-S] dans les résultats de recherche, puis appuyez sur le bouton [Installer].

    L'installation commence. Lorsqu'elle est terminée, les boutons [Désinstaller] et [Ouvrir] sont affichés, puis l'icône [] (RISO PRINT-S) est ajoutée dans l'écran d'accueil.

Pour les utilisateurs de périphérique Windows RT :

  1. 1.Appuyez ou cliquez sur [Store] dans l'écran d'accueil de votre périphérique pour ouvrir Windows Store.

  2. 2.Recherchez [RISO PRINT-S].

  3. 3.Appuyez ou cliquez sur [RISO PRINT-S] dans les résultats de recherche.

    L'écran de détails apparaît.

  4. 4.Appuyez ou cliquez sur le bouton [Installer].

    L'écran [Connexion Microsoft] apparaît.

  5. 5.Entrez l'identifiant et le mot de passe de votre compte.

    Quand l'authentification est terminée, l'installation commence. Lorsqu'elle est terminée, un message de fin apparaît.

■Démarrage de RISO PRINT-S et ouverture du menu principal

Cette section décrit comment démarrer RISO PRINT-S et ouvrir le menu principal qui contient les fonctions de base de RISO PRINT-S.

Démarrage de RISO PRINT-S

Appuyez sur l'icône [] (RISO PRINT-S) sur votre périphérique.

RISO PRINT-S démarre et l'écran [Accueil] apparaît.

  • Sur un périphérique iOS ou Android, les documents PDF ne peuvent pas être affichés directement dans RISO PRINT-S. Vous devez d'abord ouvrir un document PDF dans une autre application d'afficheur. Pour plus d'informations, voir « Prévisualisation d’un document PDF ».

  • Lorsque RISO PRINT-S est démarré après l'ouverture d'un document PDF, l'aperçu du document s'affiche dans la zone d'aperçu de l'écran [Accueil].

Menu principal

Appuyez sur le bouton [] (menu principal) dans le coin supérieur gauche de l'écran [Accueil] pour afficher le menu principal. Chaque appui sur le bouton [] affiche ou masque successivement le menu principal.

Les fonctions de base de RISO PRINT-S sont contenues dans le menu principal.

■Enregistrement/Gestion des imprimantes

Vous devez enregistrer une imprimante avant de pouvoir l'utiliser pour imprimer des documents PDF sur RISO PRINT-S. Vous pouvez ajouter d'autres imprimantes, supprimer des imprimantes inutiles et modifier l'imprimante par défaut. Les imprimantes enregistrées peuvent être gérées selon vos besoins.

Vous pouvez enregistrer jusqu'à 10 imprimantes.

  1. 1.Mettez sous tension l'imprimante cible et le contrôleur de périphérique externe.

  2. 2.Ouvrez le menu principal dans l'écran [Accueil].

  3. 3.Appuyez sur [Imprimantes].

    L'écran [Imprimantes] s'affiche. La liste des imprimantes disponibles s'affiche si des imprimantes ont déjà été enregistrées.

  4. 4.Appuyez sur [] (Rechercher) dans le coin supérieur droit de l'écran [Imprimantes].

    L'écran [Chercher] s'affiche.

  5. 5.Appuyez sur l'icône [+] de l'imprimante cible.

    L'imprimante est ajoutée à la liste de l'écran [Imprimantes].

    Vous pouvez confirmer l'état des imprimantes à l'aide du symbole ○ à gauche des noms d'imprimante.

    Vert : En ligne

    Gris : Hors ligne

    Lorsque l'imprimante cible est introuvable :

    Vous pouvez ajouter manuellement une imprimante en définissant l’adresse IP.

    Appuyez sur [] (Ajouter) dans le coin supérieur droit de l'écran [Imprimantes]. Appuyez sur le champ d'entrée de l'adresse IP dans l'écran [Ajouter], entrez l'adresse IP du contrôleur de périphérique externe qui est connecté à l'imprimante cible, puis appuyez sur [+].

    La première imprimante ajoutée est automatiquement sélectionnée comme imprimante par défaut de RISO PRINT-S.

  6. 6.Modifiez l'imprimante par défaut selon vos besoins.

    Pour les utilisateurs de tablette :

    Appuyez sur [Yes] à côté du paramètre [Définir par défaut] de l'imprimante à définir comme imprimante par défaut dans l'écran [Imprimantes].

    L'imprimante sélectionnée devient l'imprimante par défaut et son nom est surligné en noir.

    Pour les utilisateurs de smartphone :

    Appuyez sur l'icône［］de l'imprimante à définir comme imprimante par défaut dans l'écran [Imprimantes].

    L'écran [Info sur l'impr] s'affiche.

    Appuyez sur [Yes] à côté de [Définir par défaut].

  7. 7.Configurez les paramètres d'impression par défaut selon vos besoins.

    Les paramètres d'impression à configurer ici seront ceux que l'imprimante sélectionnée utilisera par défaut. Vous pouvez aussi configurer des paramètres d'impression pour chaque document PDF à imprimer. Pour plus d'informations, voir « Configuration des paramètres d'impression dans RISO PRINT-S ».

■Prévisualisation d'un document PDF

Vous devez ouvrir un document PDF à l'aide d'une application d'afficheur avant de démarrer RISO PRINT-S.

Pour les utilisateurs iPhone ou iPad :

  1. 1.Ouvrez un document cible à l'aide d'une application d'afficheur, puis appuyez sur [] dans le coin supérieur droit de l'écran.

  2. 2.Appuyez sur [Ouvrir dans…] dans le menu.

  3. 3.Appuyez sur l'icône [] (Ouvrir dans RISO PRINT-S).

    Le document s'affiche dans la zone d'aperçu de RISO PRINT-S.

Pour les utilisateurs de périphérique Android :

  1. 1.Ouvrez un document cible à l'aide d'une application d'afficheur, puis appuyez sur [] dans le coin supérieur droit de l'écran.

  2. 2.Appuyez sur [Partager] dans le menu.

    La liste [Partager] s'affiche.

  3. 3.Appuyez sur l'icône [] (RISO PRINT-S).

    Le document partagé s'affiche dans la zone d'aperçu de RISO PRINT-S.

Pour les utilisateurs de périphérique Windows RT :

  1. 1.Appuyez ou cliquez sur [Ouvrir un document] dans l'écran [Accueil], puis ouvrez un document cible.

    Le document PDF s'affiche dans la zone d'aperçu de RISO PRINT-S.

    Pour ouvrir un autre document, fermez celui qui est actuellement affiché et accédez à l'écran [Accueil] ou fermez RISO PRINT-S à l'aide du gestionnaire de tâches et redémarrez RISO PRINT-S.

■Configuration des paramètres d'impression dans RISO PRINT-S

Vous pouvez configurer les paramètres d'impression tels que l'impression recto/verso, l'imposition, la perforation et l'agrafage dans RISO PRINT-S pour chaque document PDF ou imprimante.

Pour configurer les paramètres d'impression d'un document PDF actuellement affiché :

Configurez les paramètres d'impression d'un document PDF actuellement affiché. Les paramètres configurés demeurent en vigueur jusqu'à ce qu'un autre document soit ouvert ou que le périphérique soit mis hors tension.

  1. 1.Prévisualisez un document PDF, puis appuyez sur le bouton [] (Param. impress.).

    L'écran [Param. impress.] s'affiche. Pour plus d'informations, voir « Fonctions des paramètres d'impression ».

Pour configurer les paramètres d'impression par défaut de l'imprimante actuellement sélectionnée :

Vous pouvez configurer la valeur par défaut des paramètres d'impression pour chaque imprimante. La valeur du paramètre d'impression configurée pour l'imprimante demeure en vigueur jusqu’à ce qu'elle soit modifiée. La valeur est conservée même si le périphérique est mis hors tension.

  1. 1.Appuyez sur [Imprimantes] dans le menu principal.

    La liste des imprimantes enregistrées apparaît.

  2. 2.Appuyez sur le bouton [] (Prm d'impress par déf) de l'imprimante cible.

    L'écran [Prm d'impress par déf] s'affiche. Pour plus d'informations, voir « Fonctions des paramètres d'impression ».

■Fonctions des paramètres d'impression

Les paramètres d'impression suivants sont disponibles dans RISO PRINT-S. Les fonctions disponibles varient toutefois selon votre imprimante et vos périphériques en option.

Éléments

Contenu

Basique

Mode couleur

Permet de définir le mode couleur.

Copies

Entrez le nombre de copies à imprimer.

Recto/verso

Permet d'activer le mode d'impression recto/verso.

Format papier

Permet de sélectionner le format du papier à utiliser pour l'impression.

Ajuster au papier

Permet d'agrandir/réduire les impressions pour les adapter au format sélectionné sous [Format papier].

Type papier

Permet de sélectionner le format du papier à utiliser pour l'impression.

Bac d'entrée

Permet de sélectionner l'emplacement du bac d'entrée du papier à utiliser pour l'impression.

Mise en page

Imposition

Permet de définir le nombre de pages à imprimer sur une feuille. Vous avez le choix entre [Off], [2 pages] et [4 pages].

Ordre mise en page

Permet de définir l'ordre des pages à imprimer sur une feuille.

Finition

Assembler

Permet de spécifier l'ordre de tri des impressions en copies multiples.

・Lorsque [Grouper] est sélectionné, [Agrafe] et [Livret] sont désactivés.

・Ce paramètre est disponible uniquement lorsque l'unité de finition en option est configurée sur l'imprimante.

Livret

Active ou désactive l’impression au format Livret.

・Ce paramètre est disponible uniquement lorsque l'unité de finition en option est configurée sur l'imprimante.

Reliure livret

Permet de sélectionner le style de reliure lorsque l'option [Livret] est activée.

Vous pouvez aussi configurer l'agrafage et le pliage.

[Off] : désactive la reliure livret.

[Pliage papier] : active le pliage du papier en deux à l'impression.

[Pli + Agrafe] : active le pliage du papier en deux et l'agrafe à l'impression.

Ordre mise en page

Permet de sélectionner l'ordre des pages lorsque l'option [Livret] est activée.

Côté reliure

Permet de sélectionner la position de reliure à agrafer ou à perforer.

Agrafe

Active ou désactive l'agrafage des impressions. La position d'agrafage varie en fonction du paramètre [Côté reliure].

・Ce paramètre est disponible uniquement lorsque l'unité de finition en option est configurée sur l'imprimante.

Perforation

Active ou désactive la perforation des impressions.

・[Perforation] n'est pas disponible avec [Reliure livret].

・Ce paramètre est disponible uniquement lorsque l'unité de finition en option est configurée sur l'imprimante.

Bac de sortie

Permet de sélectionner l'emplacement du bac de réception des copies imprimées.

Lorsque vous sélectionnez [Auto], les copies finies sont automatiquement empilées dans le bac approprié. Sélectionnez [Auto] en mode d'utilisation normale.

・Ce paramètre est disponible uniquement lorsque l'unité de finition en option est configurée sur l'imprimante.

Authentification

Impression sécurisée

Permet de spécifier si un code NIP doit être affecté au travail d'impression à partir de RISO PRINT-S.

Les travaux auxquels un code NIP est affecté sont à l'état [En attente] (travaux en attente de commandes). Pour imprimer des travaux auxquels un code NIP est affecté, entrez le code NIP sur le panneau de commande de l'imprimante.

Code NIP

Entrez un code NIP de huit chiffres maximum.
