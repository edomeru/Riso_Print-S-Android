■Introduction

RISO PRINT-S is an application to print PDF documents directory from tablets or smartphones.

As a matter of course you can print PDF documents saved in your devices, you can also configure several print settings like duplex print, imposition, punch and staple by using RISO PRINT-S.

This guide describes how to install, use, and setup this application, and some restrictions. Read this guide before/during use of this application, or when you find something not to clear. Please make this guide useful for a long time with RISO PRINT-S.

■Table Of Contents

■Usage Environment

System Configuration

Wireless LAN (Wi-Fi) connection environment is required beforehand.

Please refer to the instructions prepared for your wireless LAN (Wi-Fi) environment to connect and configure it.

Supported Languages

Japanese, English, French

Printable Data

PDF*1versions 1.3, 1.4, 1.5, 1.6, 1.7*2

*1: Encrypted PDF documents are not supported.

*2: Extended features are not supported.

Output Destinations

Printing can be output to the following ComColor FW/GD/FT Series printers with the PS Kit FG10 installed.

■Installing RISO PRINT-S

Install RISO PRINT-S on your tablet or smartphone.

  1. 1.Open [Google PlayTM] on your device, and search for [RISO PRINT-S].

  2. 2.Tap the [RISO PRINT-S] in the search results, and tap the [Install] button.

    Installation starts. When it is completed, the [Uninstall] and [Open] buttons are displayed, and [] (RISO PRINT-S) icon is added on the home screen.

■Starting RISO PRINT-S and Opening the Main Menu

This section describes how to start RISO PRINT-S and open the main menu that contains the basic functions of RISO PRINT-S.

Starting RISO PRINT-S

Tap [] (RISO PRINT-S) icon on your device.

RISO PRINT-S starts and the [Home] screen appears.

  • Please open a PDF document with other viewer application beforehand. For details, see “Previewing a PDF document”.

  • When RISO PRINT-S is started after opening a PDF document, the document preview is displayed on the preview area of the [Home] screen.

Main Menu

Tap the [] (main menu) button on the upper left corner of the [Home] screen to display the main menu. Tapping the [] button switches the main menu to be displayed or hidden.

The basic functions of RISO PRINT-S are provided on the main menu.

Items

Contents

①

Home

Shows the [Home] screen. When a PDF document has already been opened, the document preview is displayed.

②

Printers

Shows the [Printers] screen. You can manage the printers (search, register, add or delete printers, set as default, change the default values of print settings, or SNMP community name). For details, see "Registering/Managing Printers".

③

Print Job History

Shows the [Print Job History] screen. You can confirm printed jobs and the status results. Up to 100 jobs for each printer can be displayed. You can also delete unnecessary jobs in the job list.

You can confirm print job status with the following icons.

　Indicates completed Jobs

　Indicates jobs with errors

④

Settings

Shows the [Settings] screen. You can specify the owner name. The owner name specified in this screen will be displayed on the [Owner] column in the RISO Console.

You can enter a name up to 20 characters long.

⑤

Help

Shows the [Help] screen. You can refer to the operations, settings, function details, and FAQ information of RISO PRINT-S.

⑥

Legal

Shows the [Legal] screen. You can confirm the license, trademark credit, and current version of RISO PRINT-S.

■Registering/Managing Printers

You have to register a printer to be used for printing PDF documents on RISO PRINT-S beforehand. You can add other printers, delete unnecessary printers, and change the default printer setting. The registered printers can be managed as required.

Up to 10 printers can be registered.

  1. 1.Turn on the power of the target printer.

  2. 2.Open the main menu on the [Home] screen.

  3. 3.Tap [Printers].

    The [Printers] screen appears. The list of available printers is displayed when printers have already been registered.

  4. 4.Tap [] (Search) on the upper right corner of the [Printers] screen.

    The [Printer Search] screen appears.

  5. 5.Tap [+] of the target printer.

    The printer is added in the list of [Printers] screen.

    You can confirm the status of printers with ○ on the left side of the printer names.

    Green: Online

    Gray: Offline

    When the target printer cannot be found:

    You can manually add a printer by specifying the IP address.

    Tap [] (add) on the upper right corner of the [Printers] screen. Tap the input field of the IP address on the [Add] screen, and enter the IP address of the target printer, then tap [+].

    The first added printer is automatically set as default of RISO PRINT-S.

  6. 6.Change the default printer as required.

    For tablet users:

    Tap [Yes] on [Set as Default Printer] of the printer to be set as default on [Printers] screen.

    The selected printer is set as default and the printer name is highlighted in black.

    For smartphone users:

    Tap［］of a printer to be set as default on [Printers] screen.

    [Printer Info] screen appears.

    Tap [Yes] on [Set as Default Printer].

  7. 7.Change the SNMP community name as required.

    The community name for SNMP communication will be set to "public" when the default value is not changed.

    To change the default value, tap [] on the upper right corner of the [Printers] screen.

    Alphanumeric characters (upper case/lower case) and the following symbols can be entered:

    , . / : ; @ [ ¥ ] ^ _

  8. 8.Configure the default print settings as required.

    The print settings to be configured here will be the default print settings owned by the selected printer. You can also configure print settings for each PDF document to be printed. For details, see "Configuring the Print Settings on RISO PRINT-S".

■Previewing a PDF document

You have to open a PDF document with a viewer application before starting RISO PRINT-S.

  1. 1.Open a target document with a viewer application, and tap [] on the upper right corner of the screen.

  2. 2.Tap [Share] in the menu.

    The [Share] list appears.

  3. 3.Tap [] (RISO PRINT-S) icon.

    The shared document is displayed on the preview area of RISO PRINT-S.

■Printing a PDF document from RISO PRINT-S

  1. 1.Preview a PDF document, and tap [] (Print Settings）on the upper right corner of the screen.

    The [Print Settings] screen appears.

  2. 2.Select a target printer, and tap the [Print] button.

    A pop-up of printing progress appears to confirm current processing status.

■Table Of Contents

■Configuring the Print Settings on RISO PRINT-S

You can configure the print settings like duplex print, imposition, punch, or staple on RISO PRINT-S for each PDF document or printer.

To configure the print settings of a PDF document currently displayed:

Configure the print settings for a PDF document currently displayed. The configured settings remains until other document is opened or the device power is turned off.

  1. 1.Preview a PDF document, and tap the [] (Print Settings) button.

    The [Print Settings] screen appears. For details, see "Features of Print Settings".

To configure the default print settings of the printer currently selected:

You can configure the default value of the print settings for each printer. The print setting value configured for the printer remains until it is changed. The value is not released even if the device power is turned off.

  1. 1.Tap [Printers] on the main menu.

    The list of registered printers appears.

  2. 2.Tap the [] (Default Print Settings) button of the target printer.

    The [Default Print Settings] screen appears. For details, see "Features of Print Settings".

■Table Of Contents

■Features of Print Settings

The following print settings are provided on RISO PRINT-S. However, the available features vary depending on your printer and optional devices.

Items

Contents

Basic

Color Mode

Sets the color mode.

Copies

Enter the number of copies to be printed.

Duplex print

Sets to the duplex printing mode.

Paper Size

Selects the paper size to be used for printing.

Scale to Fit

Expands/reduces printouts to fit the size set in [Paper Size].

Paper Type

Selects the paper type to be used for printing.

Input Tray

Selects the input tray location of the paper to be used for printing.

Layout

Imposition

Sets the number of pages to be printed on a sheet. You can select from [Off], [2 pages] and [4 pages].

Layout order

Selects the order of pages to be printed on one sheet.

Finishing

Collate

Sets how printouts will be sorted for multiple copies.

・When [Group] is selected, [Staple] and [Booklet] will be disabled.

Booklet

Sets whether to impose and print out in booklet format.

Booklet binding

Selects the binding style when [Booklet] is set to on.

You can also set the staple and folding.

[Off]: Does not set the booklet binding.

[Paper folding]: Outputs the paper with folding in half.

[Fold + Staple]: Outputs the paper with folding in half and stapled.

・This setting is only available when the optional finisher is set on the printer.

Imposition direction

Selects the order of pages when [Booklet] is set to on.

Binding side

Selects the binding position to be stapled or punched.

Staple

Sets whether to stale printouts. The stapled position differs depending on the setting of [Binding side].

・This setting is only available when the optional finisher is set on the printer.

Punch

Sets whether to punch the printouts.

・[Punch] cannot be set with [Booklet binding].

・This setting is only available when the optional finisher is set on the printer.

Output Tray

Selects the tray location of the finished copies.

When you select [Auto], the finished copies are automatically output to the suitable tray. Please select [Auto] for normal use.

・This setting is only available when the optional finisher is set on the printer.

Authentication

Secure Print

Sets whether to assign a PIN code to the print job from RISO PRINT-S.

Jobs with a PIN code are treated as waiting jobs (jobs awaiting commands). To print jobs with a PIN code, enter the PIN code on the operation panel of the printer.

PIN Code

Enter a PIN code of up to eight digits.
